# Hungry.com - Food Ordering Website

A responsive front-end web project for a food ordering service, designed with HTML5 and CSS3.

## 🚀 Features

- **Responsive Navigation**: Flexbox-based navbar with hover effects and logo alignment.
- **Hero Section**: Immersive landing area with a background image overlay and "Order Now" call-to-action.
- **Services Showcase**: Layout displaying services like Table Booking, Food Ordering, and Bulk Ordering.
- **Client Section**: Flexbox row featuring partner logos (e.g., KFC, McDonald's).
- **Contact Form**: Styled input fields and text area for customer inquiries.
- **Modern Styling**: Utilizes CSS variables, absolute positioning for overlays, and Google Fonts (Baloo Tammudu 2, Bree Serif).

## 🛠️ Technologies

- **HTML5**: Semantic structure.
- **CSS3**: Flexbox, Box Model, Pseudo-elements (`::before`), and custom properties.

## 📂 Project Structure

- `main.html`: The main structure of the landing page.
- `main.css`: Core styling and layout logic.
- `phone.css`: Responsive adjustments for mobile devices.

## 🏃‍♂️ How to Run

1. Clone the repository.
2. Open `main.html` in any modern web browser.
